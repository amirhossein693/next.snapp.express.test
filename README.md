# Snapp.express Product List

snapp express product list with category filter, sort and pagination

with ssr support

this project created with https://nextjs.org/

## Requirements

Node.js 12.22.0 or later<br />
NextJs<br />
Yarn

## start and test

1. Clone project
2. Go to project directory
3. Run `yarn`
4. Run `yarn next dev`
